Student Tracker

Welcome! Type a command to get started. Examples:
  • /add-student John - Add a student
  • /grade John 90 --subject Math - Add a grade
  • /behavior John positive --note "Helped peer" - Record behavior
  • /attendance John present - Mark attendance
  • /report John - Get student report
Quick Reference: /add-student <name> | /grade <name> <score> | /behavior <name> <type> | /attendance <name> <status> | /report <name>
